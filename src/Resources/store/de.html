Das Plugin bietet eine schnelle Möglichkeit Ihren Shop um Zusatz-Infos (EAN Nummer, Herstellernummer, Gewicht, Länge, Breite und Höhe) der Artikel zu erweitern und somit für die Suchmaschinen die passenden Daten zu liefern.
NEW!!!

Die Sortierung kann nun in der Konfiguration geändert werden.
Gefällt dir das Plugin oder hast du Vorschläge für unsere Plugins, dann hinterlasse uns doch eine Bewertung :)
In der Pluginkonfiguration wird folgendes eingestellt:

  • EAN anzeigen
  • Herstellernummer anzeigen
  • Gewicht anzeigen
  • Länge anzeigen
  • Breite anzeigen
  • Höhe anzeigen
  • Sortierung ändern

Die Beschriftung des Feldes lässt sich dann auch noch in den Shopware Textbausteinen individuell anpassen.

Anleitung

  • Plugin installieren
  • Pluginaktivieren
  • Plugin im Plugin Manager konfigurieren (Zusatz-Info aktivieren)
